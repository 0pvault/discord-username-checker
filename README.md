# 📛 Discord Username Checker

A simple Python script to check the availability of a Discord username using the pomelo-attempt endpoint.

# 🚀 Features
Check if a username is available on Discord
Uses Discord's internal API endpoint
Fast and lightweight
Easy to use

# 🧰 Requirements
Python 3.x
requests library

Install dependencies with:
pip install requests

# ⚙️ Usage
Clone the repository:
```md
 https://github.com/0pvault/discord-username-checker.git
 cd discord-username-checker
```

Run the script:
python script.py
